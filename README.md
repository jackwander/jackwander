## 🚀 Featured Projects

### 🪦 [Graveyard Analytics](https://github.com/jackwander/jellyfin-graveyard-analytics)

A plugin for **Jellyfin** that helps data hoarders manage their server’s "afterlife."

Features include:

- Tracking **unwatched media**
- Monitoring **real-time playback activity**
- Detecting **inactive or abandoned content**
- Automatically generating **"Leaving Soon" collections**

Designed to help media server owners reclaim **terabytes of unused storage** while understanding how their libraries are actually used.

### 📦 [Laravel Module Maker](https://github.com/jackwander/Laravel-Module-Maker)

It keeps projects **clean, organized, and scalable** by encouraging modular architecture and reducing repetitive setup work.
